Moderation Guide

Coming soon. This guide will help organizations moderate their civic engagement platforms.

Built-in Moderation Features

  • Flagging system: Users can flag inappropriate content
  • Rate limiting: Prevents spam and abuse
  • Role gates: Different permission levels for different users
  • Profanity filters: Basic content filtering (configurable)
  • Audit trail: All moderation actions are logged

Best Practices

  • Set clear community guidelines
  • Respond to flagged content within 24 hours
  • Use anonymous modes for sensitive topics
  • Regularly review and update moderation settings
