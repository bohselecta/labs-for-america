FAQ

Do I need a developer?

No. Deploy to GitHub Pages, click the ⚙️ admin button, and fill in your org name/email/colors.

Can I preview before deploying?

Yes—use the "View Template" button on any template card.

What is Fireproof?

Fireproof is a local-first database that stores data in your browser and syncs across devices. It's like having a spreadsheet that works offline and automatically saves. No server required—your data stays private and secure.

What if my organization already has internal systems?

Use the templates to run time-boxed campaigns. Export CSV/JSON and attach the files to your ticketing/records system. Larger orgs can swap Fireproof for a POST to your secure ingestion endpoint—see each template's README "Integration Touchpoints."
