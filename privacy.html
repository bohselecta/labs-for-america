Key Principles

  • Consent-first: All data collection requires explicit user consent
  • Anonymous by default: Users can participate without providing personal information
  • Minimal data: We only collect what's necessary for the civic engagement purpose
  • Local storage: Data stays in the user's browser by default using Fireproof
  • Export control: Organizations can export and delete data at any time
