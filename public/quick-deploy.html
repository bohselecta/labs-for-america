Quick Deploy – Your Template

Goal: put your portal online in ~20 minutes, using only a web browser.

This page is a condensed version of the full docs at /docs/deploy.md.

1

Create a GitHub account

Go to github.com/join and sign up. Verify your email.

2

Create a new repository

  • Click the + (top-right in GitHub) → New repository.
  • Name it, e.g. civiclabs or justicelabs. Set Public → Create.
3

Upload this template's files

  • Open your new repo → Add file → Upload files.
  • Drag the unzipped files from this template folder into the page (not the .zip itself).
  • Scroll down → Commit changes.
Open "New repository"
4

Deploy on Vercel

  • Go to vercel.com/signup and sign up with GitHub.
  • Add New → Project → select your repo → Deploy.
  • After the build, your site is live at yourname.vercel.app.
Deploy on Vercel Full deploy guide
5

Brand & Admin (optional)

  • Visit /admin → upload your logo. The site auto-themes with your colors.
  • Pick a preset: City, Police (PD), School, or Health.
  • Create your first Lab in Admin → New Lab (prize, deadline, description).
6

JusticeLabs extras (if applicable)

  • Open /admin/crisis for Manhunt / Emergency mode & public chat.
  • Use /tips for confidential tips (private channel).
